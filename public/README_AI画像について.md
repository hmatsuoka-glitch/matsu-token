## 画像サイズについて

- 全画像とも1024px幅のJPEG。解像度は4K/ヒーロー用途としては物足りない可能性あり。
- より大きな解像度が必要な場合は、AI画像の「アップスケーラー」ツール（Topaz Gigapixel, Upscayl 等）で2〜4倍に引き伸ばすことを推奨。

## 注意事項と使用上の推奨

1. **AI生成画像の明記を検討**：HPフッターや会社概要ページで「イメージ画像」と明記するのが望ましい（一般的な慣行）。
2. **実写への置き換えを推奨**：長期運用を考えると、実際の職人・現場写真に置き換えることを強く推奨。AI画像は仮運用として位置づけるのが安全です。
3. **顔の細部**：AI生成画像の顔は実在しない合成顔のため、プライバシー問題は発生しませんが、不自然な手指・装備などがあれば差し替えてください。
4. **ハルシネーション対策**：作業着や道具に実在しないブランドロゴらしきものが写り込んでいないか、必ず目視確認してください。
